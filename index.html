Latest Projects

Bikeshare user prediction

Building and Training Neural Network to predict the number of bikeshare users on a given day.

Under Development

Bike sharing systems are new generation of traditional bike rentals where whole process from membership, rental and return back has become automatic. Data being generated from this is of great importance for reseach and this virtual sensor is useful for keeping a check on the mobility of city by which important events can be monitored in the city.

In this we build a Neural Network by implementing Gradient Descent and Backpropagation and train that network on Bikeshare data for the prediction of number of users on a particular day. The dataset has various attributes such as date, season, year, month, hour, holiday, day of the week, windspeed etc. that drives the prediction.
